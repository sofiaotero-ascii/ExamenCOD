##Cloné el repositorio:
Usé el comando git clone para copiar el repositorio de GitHub a mi PC
##Entré a la carpeta del repositorio:
Navegué al directorio del repositorio con el comando cd ExamenCOD.

##Creé una nueva rama:
Creé una nueva rama llamada readme usando git checkout -b readme, para poder trabajar en ella sin afectar la rama principal.

##Modifiqué el archivo README.md:
Abrí el archivo README.md con el editor nano y realicé los cambios necesarios.

##Guardé y confirmé los cambios:
Usé git add README.md para preparar el archivo para el commit y luego ejecuté git commit -m "Añadir archivo README.md" para guardar esos cambios en la rama.
